# Customer Revenue Recovery Platform

SQL project focused on customer revenue recovery for a SaaS business.

## Technologies

- PostgreSQL
- DBeaver
- VS Code
- GitHub

## Project Structure

- `sql/01_schema.sql`: database schemas
- `sql/02_tables.sql`: core tables
- `sql/03_inserts.sql`: sample data
- `sql/04_views.sql`: analytical views
- `sql/05_audit_trigger.sql`: audit log and trigger
- `sql/06_test_queries.sql`: validation queries

## Main Features

- Customer and subscription management
- Payment tracking
- Revenue recovery KPI
- Customer health analysis
- Audit log for recovered payments
